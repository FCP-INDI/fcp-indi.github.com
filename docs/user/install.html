Installing C-PAC¶

System Requirements¶

C-PAC is designed to run in a Unix-like environment such as Linux or Mac OS X, and thus does not support Windows.

Many of the analyses carried out by C-PAC are computationally intensive. Although C-PAC should run on most computers made in the past few years, we highly recommend using a system with multiple cores or multiple processors.

Install Python Dependencies¶

Python is the glue (Bloom, 2012) that ties all the pieces of C-PAC together. Though many computers come with Python pre-installed, C-PAC relies on a number of special-purpose packages. While it is possible to install all of these packages manually, we strongly recommend that you instead use the Enthought Python Distribution (EPD).

If you have a .edu email address:
Enthought provides the full version of EPD at no cost to academic users. This includes all of the packages required by C-PAC.

If you do not have an .edu email address:
Many of the required packages are included in EPD Free, though users must still manually install NetworkX and Setuptools (see below)

If you would rather install packages manually, below is the list of C-PAC depeendencies:

  • SciPy and NumPy
  • NetworkX
  • Traits
  • iPython
  • Matplotlib
  • Setuptools

WARNING: Manually installing the above dependencies is a very involved process, and is beyond the scope of this guide. If you are not familiar with the process of installing python packages, we suggest you seek assistance from someone who is. Future versions of C-PAC will include tools to do this automatically.

Install Nipype and Nibabel¶

C-PAC relies heavily on tools developed as part of the Neuroimaging in Python (NiPy) project. Specifically, Nipype and Nibabel must be installed for C-PAC to run.

To install Nibabel, run the following command in terminal:

easy_install nibabel

C-PAC currently requires a custom version of Nipype, which which is available on the FCP-INDI Github Nipype page. Click the button labeled “Download as tar.gz”.

Open a terminal window and unpack the file using the following command:

tar -xzvf filename.tar.gz

Where filename.tar.gz is the name of the file you just downloaded.

This will result in a new directory containing Nipype files. Navigate into this directory in terminal and type:

sudo python setup.py install

You will be asked for the root password.

Install AFNI and FSL¶

C-PAC harnesses the power of two leading neuroimaging software packages to perform key analyses:

  • AFNI (which stands for Analysis of Functional NeuroImages) is a set of programs for processing, analyzing and displaying fMRI data. If you are using Mac OS X 10.7 or greater, follow these instructions.
  • FSL (which stands for FMRIB Software Library) is a comprehensive library of analysis tools for FMRI, MRI, and DTI brain imaging data.

Warning: C-PAC has only been tested with FSL version 4.1, and may not yet support the recently released version 5.0. We are working to port and test C-PAC on FSL 5.0. For now, users can still download old versions of FSL.

Editing .bashrc¶

If you have not already done so, make sure the paths to FSL and AFNI have been added to your .bashrc file. To do this, open a new terminal window and run:

nano ~/.bashrc

This will open .bashrc in the nano text editor. Scroll to the bottom of the file and add the following text:

# Path to FSL
FSLDIR=/path/to/fsl
. ${FSLDIR}/etc/fslconf/fsl.sh
PATH=${FSLDIR}/bin:${PATH}
export FSLDIR PATH

# Path to AFNI
export PATH=$PATH:/path/to/afni

Where /path/to/fsl and /path/to/afni are the locations where FSL and AFNI are installed (usually /usr/local/fsl for FSL and ~/abin for AFNI). Save .bashrc and exit nano.

If these changes have worked, you should be able to launch FSL and AFNI from the terminal simply by typing afni and fsl.

NOTE: If you are using a different terminal, such as csh or tcsh, follow the equivalent steps for those platforms.

Install FreeSurfer (optional, but required for vertex timeseries extraction)¶

FreeSurfer is a set of automated tools for reconstruction of the brain’s cortical surface from structural MRI data, and overlay of functional MRI data onto the reconstructed surface. Click here for instructions on how to download and install FreeSurfer.

Users wishing to run vertex timeseries extraction must also install the gradunwarp python package. To do so, run the following command in terminal:

svn checkout http://gradunwarp.googlecode.com/svn/trunk/ gradunwarp

This will download the latest version of the software and put it in a directory called gradunwarp. Navigate into this directory in terminal.

If you have installed EPD, run the command:

sudo python setup.py install

If you have not installed EPD, run the command:

sudo python setup.py install --install-scripts=/usr/local/bin

In both cases, you will be asked for the root password.

Install C-PAC¶

Congratulations, you are now ready to install C-PAC itself!

C-PAC is available for download from the C-PAC Homepage. Click the button labeled “Download as tar.gz”.

Open a terminal window and unpack the file using the following command:

tar -xzvf filename.tar.gz

Where filename.tar.gz is the name of the file you just downloaded.

This will result in a new directory containing C-PAC. Navigate into this directory in terminal and type:

sudo python setup.py install

You will be asked for the root password.

C-PAC will be installed along side other python packages , and should now be ready to run. If you wish, you may now safely delete the setup files downloaded during the install process.
