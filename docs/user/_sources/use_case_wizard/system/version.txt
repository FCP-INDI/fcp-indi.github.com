How to pull C-PAC image?
#######################


**C-PAC Versions** 

Current available versions: |Versions|, development (not stable)


**Docker**

To pull the latest version of C-PAC from Docker Hub:

.. code-block:: none

    docker pull fcpindi/c-pac:latest

To pull a different version, please specify the image tag. E.g. fcpindi/c-pac:release-v1.5.0 is for C-PAC version 1.5.0; fcpindi/c-pac:nightly is C-PAC development version (not stable).


**Singularity**  

To pull the latest version of C-PAC from Singularity Hub:

.. code-block:: none

    singularity pull --name C-PAC.sif shub://FCP-INDI/C-PAC
