Installing CPAC

.. _install_cpac:

Downloading the latest version of CPAC
======================================

We no longer support local installations. Rather, we recommend working in a container. To get the latest Singularity or Docker container, see `Quickstart <../../user/quick.html>`_ or build your own from the `source code <https://github.com/FCP-INDI/C-PAC>`_.
